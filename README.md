##他人賞賛アプリ

### デプロイ
https://boiling-meadow-17941.herokuapp.com/

### 起動手順
```
git clone git@github.com:Ryo-ta8976/praise_message_app.git
cd praise_message_app
npm i
npm start
```

### 機能
- メッセージ投稿機能
- ユーザー選択機能
- メッセージ一覧表示機能
- 自分と関係のない投稿に対する拍手機能(拍手ボタン)
- 拍手回数履歴機能(マウスホバー時)
- 拍手回数カウント機能
